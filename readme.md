## How to install

- Clone this repo

  ```sh
  git clone git@github.com:pataruco/ga.git
  ```

- Install dependencies

  ```sh
  yarn
  ```

- Boostrap dependencies

  ```sh
  yarn lerna bootstrap
  ```

## Deployment
